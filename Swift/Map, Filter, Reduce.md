## 고차함수

<br/>

Argument 로 함수를 받거나,

Return 값을 함수로 return 하는 함수

<br/>

<br/>

### argument vs parameter

```swift
func sum(a: Int, b: Int) -> Int { //parameter
    return a + b
}

sum(a: 1, b: 2) //argument

```

Argument 전달인자, 값을 전달해준다.

Parameter 매개변수, argument 의 값을 받아들이는 변수
