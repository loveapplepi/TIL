## Object-C

Objective-C 소스 파일은 .m 파일 이름의 확장자로 한다.

자주 사용되는 파일 확장자 목록

| 확장자    | 의미                     |
| --------- | ------------------------ |
| .c        | C 언어 소스 파일         |
| .cc, .cpp | C++ 언어 소스 파일       |
| .h        | 헤더 파일                |
| .m        | Objective-C 소스파일     |
| .mm       | Objective-C++ 소스파일   |
| .pl       | Perl 소스 파일           |
| .O        | (컴파일된) 오브젝트 파일 |
